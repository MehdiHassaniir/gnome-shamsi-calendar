#### Install , نصب:
> * Clone the repository (Download) , دریافت مخزن
> * Go to the cloned directory , رفتن به پوشه‌ی آن
> 
> * 1- run: ```./1-install.sh``` :اجرای
> 
> * 2- Restart gnome-shell , اجرای مجدّد گنوم:\
> xorg => (Alt+F2 -> r -> Enter)\
> xorg or wayland => (logout + login)
> 
> * 3- run: ```./2-install.sh``` :اجرای
> * OK , کار تمام است
